# 一、项目描述
### 项目基于 Vue 全家桶 + ES6 + Webpack 等技术，采用模块化、组件化、工程化的模式开发; 主要包含商城主页，购物车结算页，商品分类页，个人中心、商品详情页等功能模块，其中首页包含商品轮播图、首页推荐、本周流行、商品长列表展示等子模块，商品详情展示了商品和店铺信息、商品详情图片、用户评论等内容。
